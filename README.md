# LAN Scanner

Local network scanner server, enabling you to kick unwanted or unknown MAC, with WEB GUI

![alt text](https://github.com/thomasRousseau/LAN_Scanner/blob/master/appearance.png "Appearance")

#### Requirements :
* Python3.4
* Flask
* Sqlite3
